# 🗓️ ITIC Club & Learning Roadmap

This document outlines the structured learning milestones and activities planned for the club. Our roadmap is designed to take members from complete beginners to industry-ready developers and innovators.

---

## 🚀 Phase 1: Foundation (Weeks 1-4)
Focus on building core programming skills, version control, and basic terminal usage.
- [ ] **Git & GitHub Basics**: Version control, branching, pull requests, and open source.
- [ ] **Programming Fundamentals**: Introduction to Python and JavaScript.
- [ ] **Command Line Interface**: Navigating the terminal, file system commands.

## 💻 Phase 2: Specialization Tracks (Weeks 5-10)
Members split into specialized pathways based on interests:
- **Web Development**: HTML/CSS, Vanilla JS, and modern frontend frameworks (React, Next.js).
- **Mobile Development**: Flutter and React Native.
- **AI & Data Science**: NumPy, Pandas, Scikit-Learn, and introductory neural networks.
- **Cybersecurity & Networking**: Basic networking, Linux administration, and security fundamentals.

## 🛠️ Phase 3: Project Building & Collaboration (Weeks 11-14)
Applying skills to build collaborative projects:
- [ ] **Internal Hackathons**: Weekend-long coding sessions focused on solving local CUT problems.
- [ ] **Group Projects**: Collaborative building using Agile methodologies.
- [ ] **Portfolio Reviews**: Code reviews and optimization sessions.

## 💼 Phase 4: Career & Industry Readiness (Ongoing)
Transitioning from student to professional:
- [ ] **Resume & LinkedIn Optimization**
- [ ] **Mock Interviews**: LeetCode practices, system design, and soft-skill prep.
- [ ] **Tech Talks**: Guest speakers from the tech industry.
