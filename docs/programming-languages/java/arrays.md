# 📊 Arrays in Java

Arrays are fixed-size containers holding elements of the same data type.

---

## 💻 Declaring & Accessing Arrays
```java
// Declaration & Allocation
int[] numbers = new int[5];
numbers[0] = 42; // Index starts at 0

// Literal Allocation
int[] scores = {90, 85, 95};

// 2D Array (Matrix)
int[][] matrix = {
    {1, 2},
    {3, 4}
};
```
