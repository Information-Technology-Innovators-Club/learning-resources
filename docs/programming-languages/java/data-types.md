# 🗄️ Java Data Types

Java has two main categories of data types: Primitive and Reference.

---

## 1. Primitives (Stored in Stack Memory)
* `byte` (8-bit)
* `short` (16-bit)
* `int` (32-bit integer)
* `long` (64-bit integer, suffix `L`)
* `float` (32-bit floating point, suffix `f`)
* `double` (64-bit floating point)
* `char` (16-bit Unicode character, single quotes `'A'`)
* `boolean` (`true` or `false`)

## 2. Reference Types (Stored in Heap Memory)
Points to objects. Examples: `String`, `Array`, `Class`, interface instances.
