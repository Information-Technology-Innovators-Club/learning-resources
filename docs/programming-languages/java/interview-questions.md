# 💼 Top Java Interview Questions

Use these questions to prepare for technical interviews.

---

### Q1. What is the difference between JVM, JRE, and JDK?
* **JVM (Java Virtual Machine)**: Runs Java bytecode.
* **JRE (Java Runtime Environment)**: JVM + libraries to run Java apps.
* **JDK (Java Development Kit)**: JRE + tools (javac, debugger) to build Java apps.

### Q2. Why are String objects immutable?
For security, caching in the String Pool, and thread safety.

### Q3. What is the difference between `==` and `.equals()`?
- `==` compares reference memory locations.
- `.equals()` evaluates content equality.
