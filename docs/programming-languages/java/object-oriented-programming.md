# 🏛️ Object-Oriented Programming (OOP) in Java

Java is centered around OOP principles. OOP organizes software design around objects rather than logic rules.

---

## 🏆 The 4 Core Pillars

### 1. Encapsulation
Restricting direct access to object state by using `private` variables and exposing `public` getters/setters.

### 2. Inheritance
A class acquiring properties and methods of another class using the `extends` keyword.
`class Developer extends Member {}`

### 3. Polymorphism
Performing a single action in different ways. Includes Method Overriding (runtime) and Overloading (compile-time).

### 4. Abstraction
Hiding internal details and showing only essentials using `abstract` classes and `interfaces`.
