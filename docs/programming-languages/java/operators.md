# ⚡ Operators in Java

Operators are symbols used to perform operations on variables and values.

---

## 🏷️ Operator Classes

| Class | Operators | Example |
| :--- | :--- | :--- |
| **Arithmetic** | `+`, `-`, `*`, `/`, `%`, `++`, `--` | `int sum = a + b;` |
| **Relational** | `==`, `!=`, `>`, `<`, `>=`, `<=` | `boolean equal = a == b;` |
| **Logical** | `&&` (AND), `\|\|` (OR), `!` (NOT) | `boolean valid = check && exist;` |
| **Assignment** | `=`, `+=`, `-=`, `*=`, `/=` | `a += 5;` |
| **Bitwise** | `&`, `\|`, `^`, `~`, `<<`, `>>` | `int result = a & b;` |
