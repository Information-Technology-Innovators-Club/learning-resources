# 📋 Java Self-Assessment Quiz

Test your basic understanding of Java logic.

---

### Q1: What is the default value of a local variable?
A) `0`  
B) `null`  
C) Local variables do not have default values; they must be initialized.  
*Answer: C*

### Q2: Which collection does not permit duplicates?
A) List  
B) Set  
C) Queue  
*Answer: B*
