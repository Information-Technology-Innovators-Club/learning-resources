# 📅 Week-by-Week Java Roadmap

Follow this 6-week study schedule to master core Java programming.

---

## 🗓️ Weekly Milestones

### 📅 Week 1: Core Fundamentals
- [ ] Variables & Primitive Types
- [ ] Operators & Arithmetic
- [ ] Input/Output via Scanner

### 📅 Week 2: Control Flow & Arrays
- [ ] If/Else & Switches
- [ ] For/While loops
- [ ] Single and Multidimensional Arrays

### 📅 Week 3: Object-Oriented Java
- [ ] Classes & Objects
- [ ] Pillars: Encapsulation, Inheritance, Polymorphism, Abstraction

### 📅 Week 4: Collections & Memory
- [ ] Lists, Sets, and Maps
- [ ] Exception Handling (Try-Catch)

### 📅 Week 5: DB Connects & File I/O
- [ ] File Streams
- [ ] JDBC connection executing query routines

### 📅 Week 6: Enterprise Boot
- [ ] Maven & Gradle build tools
- [ ] REST API mapping via Spring Boot
