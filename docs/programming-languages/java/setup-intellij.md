# 💎 Setting Up IntelliJ IDEA for Java

IntelliJ IDEA is the premier integrated development environment (IDE) for professional Java developers, created by JetBrains.

---

## 🚀 Getting Started
1. Download **IntelliJ IDEA Community Edition** (free) or Ultimate Edition from [jetbrains.com/idea](https://www.jetbrains.com/idea/).
2. Launch the IDE and open the SDK settings (`File > Project Structure > Project`).
3. Under SDK, select **Add SDK > Download JDK** and choose version 17 or 21.
4. Enable **Auto-Import** for Maven/Gradle dependencies to build code seamlessly.
