# 📝 Java Syntax Fundamentals

Java is a strongly typed, class-based object-oriented programming language. Every program starts within a class.

---

## 💻 Standard Boilerplate Structure
Every standalone Java file running code must have a class named exactly matching the filename and a `main` method entry point:

```java
public class Main {
    // Application entry point
    public static void main(String[] args) {
        System.out.println("Hello, CUT ITIC Developers!");
    }
}
```

### Key Rules:
1. **Semicolons**: Every statement must end with a semicolon `;`.
2. **Case Sensitivity**: `Main` is different from `main`.
3. **Naming Conventions**: Classes use `PascalCase`, methods and variables use `camelCase`.
