# 🌿 Git Setup & Guides

Distributed version control system to track source history.

---

## 🚀 Getting Started & CLI Setup
```bash
    git init                  # Initialize repository
    git checkout -b <branch>  # Create and switch to new branch
    git add .                 # Stage all file modifications
    git commit -m "msg"      # Save staged changes as commit
    git checkout <branch>     # Switch back to another branch
    git merge <branch>        # Merge code history locally
```

---

## 📋 Core Reference Sheet

| Command | Action | Use Case |
| :--- | :--- | :--- |
| `git status` | View modifications status | Check what is staged vs unstaged |
| `git log --oneline` | View simplified commits log | Trace commit history |
| `git stash` | Temporarily save unstaged code | Switch branch without committing |
