# 🐙 GitHub Setup & Guides

Cloud hosting platform for Git repositories, actions, and projects.

---

## 🚀 Getting Started & CLI Setup
```bash
    gh auth login             # Authenticate GitHub CLI tool
    gh repo clone <repo>      # Clone repositories via CLI
    gh pr create              # Create Pull Request from terminal
    gh issue create           # Open repository bug report
```

## 📋 Core Reference Sheet

| Feature | Description | File Trigger |
| :--- | :--- | :--- |

| **Issue Templates** | Standardize bug reports | `.github/ISSUE_TEMPLATE/` |
| **Pages** | Free static site hosting | Repo Settings |
