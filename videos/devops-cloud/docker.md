# 🐳 Docker Video Channels & Playlists

Learn to package, test, and ship isolated container applications.

## 📽️ Recommended YouTube Channels

### 1. [TechWorld with Nana](https://www.youtube.com/@TechWorldwithNana)
* **Core Specialty**: Dockerfiles, volumes, docker-compose, basic setups
* **Recommended Playlists / Videos**:
  - *Docker Tutorial for Beginners*

### 2. [DevOps Directive](https://www.youtube.com/@DevOpsDirective)
* **Core Specialty**: Caching builds, multi-stage builds, production setups
* **Recommended Playlists / Videos**:
  - *Docker Hands-on Guide*

### 3. [KodeKloud](https://www.youtube.com/@KodeKloud)
* **Core Specialty**: Local container configurations, network routing basics
* **Recommended Playlists / Videos**:
  - *Docker for Absolute Beginners*
