# 🎨 Frontend Video Channels & Playlists

Master frontend development, styling, and modern single-page-application frameworks.

---

## 📽️ Recommended YouTube Channels

### 1. [Kevin Powell](https://www.youtube.com/@KevinPowell)
* **Core Specialty**: CSS Layouts, Responsive Web Design, Flexbox/Grid
* **Recommended Playlists / Videos**:
  - *CSS Demystified*
  - *Conquering Responsive Layouts*

### 2. [Web Dev Simplified](https://www.youtube.com/@WebDevSimplified)
* **Core Specialty**: Vanilla JS concepts, React Hooks, UI State
* **Recommended Playlists / Videos**:
  - *React Hooks Simplified*
  - *Learn JS in 15 Minutes*

### 3. [Jack Herrington](https://www.youtube.com/@jherr)
* **Core Specialty**: Advanced React, Micro-frontends, State architectures
* **Recommended Playlists / Videos**:
  - *React Federation*
  - *State Management Guide*

### 4. [Traversy Media](https://www.youtube.com/@traversymedia)
* **Core Specialty**: Crash courses on JS, HTML/CSS, React, Tailwind
* **Recommended Playlists / Videos**:
  - *Modern HTML/CSS from Scratch*
  - *React JS Crash Course*
